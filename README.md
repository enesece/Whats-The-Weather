# React Weather App

A responsive weather dashboard built with React and Vite. It fetches real-time weather data for any city using the OpenWeatherMap API.

**Live Demo:** https://falon-whatstheweather.netlify.app/

## Features

- **Current Weather:** Displays temperature, wind speed, humidity, feels-like temp, visibility, and pressure.
- **Search Functionality:** dynamic search for cities worldwide.
- **Dark/Light Mode:** Toggleable theme support.
- **Responsive Design:** Optimized for all screen sizes using CSS Modules.

## Tech Stack

- **Framework:** React + Vite
- **Styling:** CSS Modules
- **API:** OpenWeatherMap API
